# Data Processing in JavaScript

## Overview
This implementation demonstrates how to perform **Data Science** tasks using modern JavaScript features. While Python is the go-to language for data science, JavaScript is increasingly used for browser-based data analysis and high-performance server-side processing.

## Core Concepts
The JavaScript code illustrates several key techniques:

1.  **Functional Programming**:
    - **`map`**: Transform raw user data into simplified summaries.
    - **`filter`**: Isolate top-performing items based on a score threshold.
    - **`reduce`**: Calculate averages and group data into categories.
2.  **Linear Regression**:
    - A simple implementation of the **Least Squares Method** for predicting values.
    - Uses an object-oriented approach with a `predict` method for future estimations.

## Techniques Demonstrated
- **Data Transformation**: How to map raw datasets to more usable formats.
- **Categorization**: Grouping data based on a specific property (e.g., category).
- **Statistical Analysis**: Calculating the mean score of a dataset.

## Mathematical Implementation
The linear regression part uses the standard formula for finding the slope ($m$) and intercept ($b$) of a line:
$$y = mx + b$$
The code calculates these coefficients from an array of $x, y$ pairs.

## How to Run
Run the code using Node.js:
```bash
node "Data Processing.js"
```
