# JavaScript Variables and Strings

## Overview
This tutorial covers how to declare variables and work with text (strings) in modern JavaScript (ES6+).

## Key Components
1.  **`const`**: Declares a variable whose value cannot be re-assigned.
2.  **`let`**: Declares a block-scoped variable whose value can be changed.
3.  **`var`**: The traditional way to declare variables (avoid in modern JS).
4.  **Template Literals**: Uses backticks (`` ` ``) and `${}` to embed expressions inside strings.

## Basic Syntax
```js
const name = "Alice";
let score = 100;

// Template Literal
const result = `Player ${name} has a score of ${score}.`;
console.log(result);
```

## Common String Methods
- **`.toUpperCase()`**: Converts string to uppercase.
- **`.charAt(index)`**: Gets character at a specific position.
- **`.slice(start, end)`**: Extracts a portion of the string.
- **`.length`**: Property that returns the number of characters in the string.
