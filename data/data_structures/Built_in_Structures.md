# Built-in Data Structures in JavaScript

## Overview
JavaScript provides several built-in structures for managing collections of data. While **Arrays** and **Objects** are the most common, ES6 introduced **Map** and **Set** for more specialized use cases.

## Key Data Structures
1.  **Array**: An ordered collection of values. JavaScript arrays are dynamic and can hold mixed types.
2.  **Object**: The fundamental building block of JS. Used to store keyed collections of various data.
3.  **Map**: A collection of keyed data items, similar to an Object. However, `Map` allows keys of any type and maintains insertion order.
4.  **Set**: A collection of unique values. Useful for removing duplicates from an array.

## Common Operations
- **Array**: `push()`, `pop()`, `map()`, `filter()`, `reduce()`.
- **Map**: `set()`, `get()`, `has()`, `delete()`.
- **Set**: `add()`, `has()`, `clear()`.

## Basic Syntax
```js
const arr = [1, 2, 3];
const obj = { key: 'value' };
const map = new Map();
const set = new Set();
```
