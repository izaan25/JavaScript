# JavaScript Programming Repository

A comprehensive collection of JavaScript projects, algorithms, and learning materials organized into a structured learning path.

## 📁 Structured Learning Path

The data is organized into numbered categories to make it easier to follow a logical learning progression.

```
JavaScript/data/
├── 01) Getting Started/      # Installation and setup
├── 02) Fundamentals/         # Basic syntax, variables, strings, and types
├── 03) Object Oriented/      # Classes, inheritance, objects, and prototype
├── 04) Data Structures/      # Arrays, maps, sets, and linked lists
├── 05) Algorithms/           # Search, sorting, and algorithmic challenges
├── 06) Mathematics/          # Mathematical computations and series
├── 07) Game Development/     # Game logic and graphics
├── 08) Web and API/          # Web development and Express.js APIs
├── 09) Data Science and ML/  # Data manipulation and machine learning
└── 10) Advanced Topics/      # Async/Await, functional programming, and more
```

## 🏁 Installation & Usage
1. **Clone the repository:**
   ```bash
   git clone https://github.com/SENODROOM/PolyCode.git
   cd PolyCode/JavaScript
   ```
2. **Run a program with Node.js:**
   ```bash
   node "data/02) Fundamentals/01) Variables and Strings.js"
   ```
